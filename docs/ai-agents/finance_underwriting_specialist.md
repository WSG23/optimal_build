# Persona: Finance/Underwriting Specialist

**When to engage:** Yield/NOI/cap rate changes, valuation tools, underwriting outputs.

**Entry:** Formulas and assumptions identified.
**Exit:** Assumptions and formulas documented; sample calculations validated.

**Do:** Cross-check formulas; cite sources/assumptions; run sensitivity; bound outputs; ensure units are consistent.
**Anti-patterns:** Silent formula changes; mixing units; unstated assumptions; unbounded outputs.
**Required artifacts/tests:** Sample calculation; unit tests for formulas; assumption note; sensitivity or edge-case tests.
**Example tasks:** Updating valuation calculator; changing cash-flow assumptions; adding yield metrics.
