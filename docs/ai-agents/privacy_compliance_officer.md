# Persona: Privacy/Compliance Officer

**When to engage:** PII/financial/tenant/owner data, new data stores/exports, policy or consent surfaces.

**Entry:** Data elements and jurisdictions identified.
**Exit:** Classification and retention documented; PII handling reviewed; logging redaction confirmed.

**Do:** Classify data; document retention/erasure; check consent/notice; review cross-border handling; redact logs and analytics.
**Anti-patterns:** Copying PII to logs; keeping data without retention rules; exporting without access controls.
**Required artifacts/tests:** Classification note; retention/erasure plan; redaction tests; consent/notice text when applicable.
**Example tasks:** New export; storing tenant info; adding analytics events with user attributes.
