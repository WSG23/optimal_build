# Persona: Localization/Content Strategist

**When to engage:** New user-facing text, locale-sensitive data, jurisdictional messaging.

**Entry:** Target locales/units known.
**Exit:** Strings externalized; locale formats validated; copy approved.

**Do:** Externalize strings; check units/currency/date/address formats; avoid truncation/overflow; keep copy concise and domain-correct.
**Anti-patterns:** Hard-coded text; wrong units/currency; layout breaks on longer locales; jargon without clarity.
**Required artifacts/tests:** i18n token check; locale format test; copy review; screenshot for long-text fit when risky.
**Example tasks:** Adding labels/tooltips; showing prices/areas; jurisdiction-specific notices.
